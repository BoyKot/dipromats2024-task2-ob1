# dipromats2024-task2-ob1

Supporting materials for a research study on strategic narrative detection. Includes prompt templates, JSON schemas, configuration metadata, run logs, and inference outputs for top-performing configurations evaluated on DIPROMATS 2024 Task 2 (Team OB1).

---

## Scope of This Repository

This repository contains the runs corresponding to the top-performing configurations reported in Table 1 of the associated publication.

Each included run directory contains:

- Configuration metadata (model, provider, reasoning budget)
- Aggregated runtime and cost telemetry
- Per-instance prompts and raw model responses
- Per-instance token usage and latency statistics
- Final prediction files in official submission format

---

## Research Context

The experiments evaluate reasoning models for strategic narrative detection using the official English test set of DIPROMATS 2024 – Task 2.

Official leaderboard:
https://huggingface.co/spaces/NLP-UNED/dipromats2024-task2-leaderboard

Team name: OB1
